# 🏋️ Fitness-Analytics-Dashboard-Power-BI

A modern and interactive Power BI dashboard designed to track fitness activities, calorie needs, membership details, and financial insights for gyms, fitness centers, and health clubs.
This dashboard provides a 360° view of client health, trainer performance, and business growth with engaging visuals and real-time analytics.

---

## 📊 Features

- ⚡ Calorie Calculator: Personalized calorie requirement calculator (BMR, TDEE, weight loss/gain goals) with age, height, weight, and activity filters.

- 🧍 Member Analytics: Insights on members by age, gender, and membership status (Active/Expired).

- 💰 Financial Dashboard: Revenue, expenses, and profit tracking with monthly breakdown.

- 🏅 Memberships Overview: Platinum, Gold, and Silver memberships with active vs expired status.

- 📈 Trainer & Client Tracking: Total clients, trainers, and their contribution to revenue.

- ⚖️ BMI (Body Mass Index): Real-time BMI calculation with weight status classification.

- 🎨 Sleek dark theme UI with modern charts, gauges, and custom KPI indicators.

---

## 🧰 Technologies Used

- Power BI Desktop
- DAX (KPIs, conditional formatting, dynamic calculations)
- Custom Measures for calorie formulas and BMI
